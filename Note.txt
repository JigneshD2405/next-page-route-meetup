Server(RSC React Server Component) vs Client Components

Vanilla React Apps Render on The client :-
Backend Server ==>(Only Returns one side Single HTML file which contains the Client-Side JS code) =>Client-side (Frontend) (Visible content is generated & rendered on the client-side (by Client side react code))

In Nextjs :-
Server Backend ==>(The Backend Executes the Server Component functions & Hence drives the to-be-rendered HTML Code) ==>Client-side (The Client-side) Receives & Render the to-be-rendered HTML Code.
